Метод some() в JavaScript используется для проверки, удовлетворяет ли хотя бы один элемент массива условию, заданному 
в виде функции обратного вызова (callback). Этот метод возвращает булево значение true, если хотя бы один элемент соответствует 
условию, и false, если ни один элемент не соответствует.
